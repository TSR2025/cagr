# Compound Interest Calculator

A friendly fintech-inspired compound interest calculator built with Next.js 15, TypeScript, Tailwind CSS, shadcn/ui components, and Recharts. It provides live projections with recurring contributions, optional one-time boosts, and exportable tables.

## Running locally

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Run the unit tests for the calculation engine:

```bash
npm test
```

## Features

- Sticky input panel with contribution and boost controls
- Live recalculated projections with debounced input updates
- Growth chart with hover details, milestone table, and full breakdown
- CSV export of year-by-year results
- Warm, approachable UI with responsive layout
